# DEMO - Robotica autonoma e resiliente

This repository collects the material needed for the challenge you have to complete

## Start Here

- Follow the official ROS 2 Humble first steps: <https://docs.ros.org/en/humble/First-Steps.html>
- Read the local LIMO Pro manual: [`Limo Pro Ros2 Foxy user manual(EN).md`](./Limo%20Pro%20Ros2%20Foxy%20user%20manual(EN).md)
- Use the course Docker setup: <https://github.com/CentroEPiaggio/docker_humble_limo>

## What Is In This Repo

- `Limo Pro Ros2 Foxy user manual(EN).md`: reference manual for LIMO Pro commands, modes, mapping, navigation, camera, and optional arm workflows.
- `DEMO.png`: demo image for the assignment/materials.


## Asking Questions

Use the repository Issues section for questions about the assignment, setup problems, or unclear steps. Include the command you ran, the error output, and what environment you are using.
